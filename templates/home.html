Face API

A simple API to a collection of faces extracted from Trove's digitised newspapers.

Currently there are about 4,500 faces from 1880 to 1911. More will be added soon.

  • Get faces
  • Get newspaper titles
  • Get years

These faces belong to people. Please respect both them and your users by providing links through to the original articles wherever possible.

The complete set of faces is also available for download.

Get faces

Parameters

  • n – number of faces (1 - 100)
  • year – year of publication
  • title_id – id of the newspaper

Examples

  • 20 random faces – /faces
  • One random face – /faces?n=1
  • 20 random faces from Table Talk – /faces?title_id=713
  • 20 random faces from Table Talk in 1907 – /faces?title_id=713&year=1907
  • 20 random faces from 1907 – /faces?year=1907

Results

Returns a randomly sorted array of face objects, with the following fields:

  • article_url – link to the newspaper article from which the face was extracted
  • date – date the article was published
  • title – title of the newspaper the article was published in
  • title_id – Trove identifier for the newspaper
  • image_url – link to the face image
  • width – width of the face image
  • height – height of the face image

Example:

      [
        {
          article_url: "http://nla.gov.au/nla.news-article71315672",
          date: "1898-07-23T00:00:00",
          height: 168,
          image_url: "http://facedepot.s3.amazonaws.com/18980723-52-71315672-1.jpg",
          title: "Australian Town and Country Journal",
          title_id: "52",
          width: 168
        }
      ]

Get newspaper titles

List all the available newspaper titles in the faces database.

Examples

  • List all titles – /titles

Results

Returns an array of newspaper titles, sorted by title.

  • title – the title of the newspaper
  • title_id – the Trove identifier for the newspaper
  • faces – the number of faces currently available from this newspaper

Example:

        {    
          faces: 185,
          title: "Adelaide Observer",
          title_id: "821"
        }

Get years

List all the available years in the faces database.

Examples

  • List all years – /years

Results

Returns an array of years, sorted by year.

  • year – the year
  • faces – the number of faces currently available from this year

Example:

        {
          faces: 447,
          year: 1901
        },
